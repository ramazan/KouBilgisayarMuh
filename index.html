Bana Acil Ulaşın - Yrd.Doç.Dr Burak İnner Binner

Java konusunda kendisini geliştirmek isteyen bir mühendis, bir öğrencinin kendisine bu konuda destek olmasını beklemekte olup aşağıdaki maili göndermiştir. Konuyla ilgilenen ve ücreti karşılığında destek verebilecekler varsa bana KONU kısmına JAVA yazarak mail gönderebilir. Mail şöyle: Java ve algoritma konusunda temel duzeyde bilgim var. Amacim mobil app yazabilmek, arduino yazilimlari yapabilmek.
